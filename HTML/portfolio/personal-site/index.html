The Time Tracker

About

The Time Tracker was my senior project during my final semester at Westminster College. My goals setting out at the start of my final semester was to create a fantastic desktop experience, a JSON mobile API, and a mobile app to interface with the API.

Tech Used

Speed was key to complete this project. I needed to develop and deploy rapidly considering the short time span for this project. These constraints affected the tools I used to build The Time Tracker.

Ruby On Rails
Ionic
A modern responsive front-end framework based on Material Design
Company Bio

We are a team of college students working on this project like it's our full time job. Any amount would help support and continue development on this project and is greatly appreciated.
